Pemrograman Dasar

Pemrograman dasar adalah fondasi utama untuk memahami bagaimana komputer menyelesaikan masalah secara otomatis. Dengan memahami konsep-konsep dasar ini, kamu siap melangkah lebih jauh ke dunia teknologi dan pengembangan perangkat lunak.

Apa Itu Pemrograman Dasar?

Pemrograman adalah proses memberikan instruksi kepada komputer agar dapat menjalankan tugas tertentu. Instruksi ini ditulis menggunakan bahasa pemrograman yang kemudian diterjemahkan oleh komputer menjadi tindakan nyata.

Pemrograman dasar mencakup konsep-konsep inti yang berlaku di hampir semua bahasa pemrograman, seperti cara menyimpan data, membuat keputusan, mengulang proses, dan mengorganisasi kode. Menguasai konsep ini berarti kamu bisa belajar bahasa pemrograman apa pun dengan lebih mudah.

Mengapa Pemrograman Dasar Penting?
  • Menjadi fondasi untuk mempelajari bahasa pemrograman apa pun
  • Melatih kemampuan berpikir logis dan sistematis
  • Membuka peluang karier di bidang teknologi
  • Membantu memahami cara kerja aplikasi dan sistem digital

Variabel & Tipe Data

Variabel adalah tempat penyimpanan data di dalam memori komputer. Bayangkan variabel seperti kotak dengan label nama — kamu bisa menyimpan nilai di dalamnya, lalu memanggilnya kembali kapan saja menggunakan nama labelnya.

02

Array

Array adalah struktur data yang menyimpan banyak nilai sekaligus dalam satu variabel. Setiap nilai memiliki indeks (posisi) yang dimulai dari angka 0. Array berguna ketika kamu perlu mengelola kumpulan data sejenis, seperti daftar nama, daftar harga, atau daftar item.

Contoh: buah = ["apel", "mangga", "jeruk"] — di sini buah[0] adalah "apel", buah[1] adalah "mangga", dan seterusnya.

03

Fungsi

Fungsi adalah blok kode yang dikelompokkan untuk menjalankan tugas tertentu. Daripada menulis kode yang sama berulang kali, kamu cukup membuat satu fungsi dan memanggilnya kapan pun dibutuhkan.

Fungsi dapat menerima parameter (nilai input) dan menghasilkan nilai kembalian (output). Contoh: fungsi hitungLuas(panjang, lebar) menerima dua angka dan mengembalikan hasil perkaliannya.

04

Percabangan (If-Else)

Percabangan memungkinkan program mengambil keputusan berdasarkan kondisi tertentu. Jika kondisi terpenuhi, jalankan perintah A; jika tidak terpenuhi, jalankan perintah B.

Struktur percabangan menggunakan kata kunci if, else if, dan else. Contoh: menentukan apakah nilai ujian lulus atau tidak, menentukan diskon berdasarkan jumlah pembelian, dan sebagainya.

05

Perulangan (Loop)

Perulangan digunakan untuk mengeksekusi blok kode yang sama secara berulang selama kondisi tertentu masih terpenuhi. Tanpa perulangan, kamu harus menulis kode yang sama ratusan kali secara manual.

Dua jenis perulangan yang paling umum: for (digunakan ketika jumlah iterasi sudah diketahui) dan while (digunakan ketika iterasi bergantung pada kondisi tertentu).

Bahasa Pemrograman untuk Pemula

Setelah memahami konsep dasar, langkah berikutnya adalah memilih bahasa pemrograman. Berikut adalah bahasa yang direkomendasikan untuk pemula beserta contoh program "Hello, World!" — program sederhana pertama yang biasanya ditulis saat belajar bahasa pemrograman baru.

🐍

Python

Sangat Mudah

Sintaks yang bersih dan mudah dibaca, cocok untuk pemula. Digunakan luas untuk data science, AI, dan web.

# Mencetak teks ke layar
print("Hello, World!")
🌐

JavaScript

Mudah

Bahasa wajib untuk pengembangan web. Bisa langsung dicoba di browser tanpa instalasi apapun.

// Mencetak teks ke konsol browser
console.log("Hello, World!");
☕

Java

Menengah

Bahasa populer di industri dan kampus. Mengajarkan konsep OOP (Pemrograman Berorientasi Objek) dengan baik.

// Struktur dasar program Java
public class Main {
  public static void main(String[] args) {
    System.out.println("Hello, World!");
  }
}
⚙️

C++

Menengah

Digunakan di sistem tertanam, game, dan kompetisi pemrograman. Mengajarkan konsep memori secara mendalam.

// Menyertakan library input/output
#include <iostream>
using namespace std;

int main() {
  cout << "Hello, World!" << endl;
  return 0;
}
🐘

PHP

Mudah

Bahasa server-side yang populer untuk membangun website dinamis. Banyak digunakan di Indonesia.

<?php
  // Mencetak teks ke halaman web
  echo "Hello, World!";
?>
📱

Kotlin

Mudah

Bahasa modern untuk pengembangan aplikasi Android. Sintaksnya lebih ringkas dibanding Java.

// Fungsi utama program Kotlin
fun main() {
    println("Hello, World!")
}
